# Job Application System

Use this system for volume applications without losing control of follow-ups.

## Weekly Target

- Send 30-50 targeted applications per week.
- Spend 2-5 minutes tailoring the first paragraph and CV variant.
- Follow up after 5 business days, then again after 10 business days when the role still looks relevant.

## CV Variant Choice

- `public/resumes/simon-lechevalier-full-stack.pdf`: default for full-stack, startup, product engineer, and general software engineer roles.
- `public/resumes/simon-lechevalier-frontend-nextjs.pdf`: React, Next.js, frontend, UI engineer, CMS, SEO, and product-facing roles.
- `public/resumes/simon-lechevalier-api-automation.pdf`: backend-light, integration, API, automation, solutions engineer, internal tools, and AI workflow roles.
- `public/resumes/simon-lechevalier-web3-integration.pdf`: Web3 frontend, wallet UX, smart-contract integration, crypto product, and blockchain roles.

## Tracking Fields

Use `docs/job-search-tracker.csv` as the working tracker. Keep status values simple:

- `Target`
- `Applied`
- `Followed up 1`
- `Followed up 2`
- `Interview`
- `Rejected`
- `Offer`

## Daily Loop

1. Add target jobs to the tracker.
2. Pick the right CV variant.
3. Apply with a short tailored message.
4. Record the application date and source.
5. Schedule follow-ups directly in the tracker.
